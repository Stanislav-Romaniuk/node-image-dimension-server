# Node Image Dimension Server

Simple HTTP server built with Node.js that validates image dimensions via URL parameters.

## 🚀 Features

- Native Node.js `http` module (no frameworks)
- URL-based dimension validation
- Configurable min/max limits
- Custom error handling
- Favicon support

## 📦 Installation
Clone repository:
